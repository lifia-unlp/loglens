# Loglens

Loglens is a lightweight, standalone, single-page web application for inspecting and live-syncing AI conversation logs (`transcript.jsonl` files) generated by systems like Antigravity and OpenCode.

## Features

- **Standalone HTML Application:** Loglens runs entirely in your browser without requiring a backend server, Node.js, or any complex installations.
- **Live Sync:** Leverages the modern File System Access API to poll local transcript files for updates in real time, making it an excellent debugging tool while a conversation is active.
- **Brain Folder Scanner:** Can scan your default AI logs directory (e.g., `~/.gemini/antigravity/brain/`) to parse all historical sessions, infer their titles based on the initial prompts, and list them in a clean UI.
- **Rich Interface:** Fully resizable panels, light/dark modes, keyboard navigation, syntax-highlighted raw JSON views, and visual badging by message type.
- **Read-Only:** Strict read-only operations. Loglens parses and visualizes your logs but will never alter or corrupt your local files.

## Usage

1. Open `index.html` in any modern web browser (Google Chrome or Edge are recommended to support the File System Access API).
2. Click **Select Brain Folder** to pick the root directory where your logs are stored (e.g., `/Users/you/.gemini/antigravity/brain`).
3. Browse the inferred conversations and select one to begin live-syncing.
4. Alternatively, use **Open Single File** to pick a specific `.jsonl` transcript.
